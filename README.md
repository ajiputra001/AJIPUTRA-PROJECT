**Kendala yang sering kita hadapi:**
Saat warung lagi ramai, kita terpaksa harus berhenti melayani, membuka HP, dan mengecek mutasi rekening terlebih dahulu hanya untuk memastikan uang pembeli sudah masuk. Proses ini ribet, memakan waktu, dan membuat antrean pembeli jadi panjang.

Berangkat dari masalah tersebut, kami mengembangkan sebuah aplikasi Android inovatif bernama **Voice-Notf** (dengan visi: *Membantu UMKM berkembang dengan teknologi*).

Aplikasi ini bekerja persis seperti QRIS Soundbox (kotak pengeras suara notifikasi yang biasanya disewa dari bank), tetapi yang ini berbentuk aplikasi HP dan **100% GRATIS!**

---

## 🌟 Apa Saja Keunggulan Aplikasi "Voice-Notf"?

- **🔊 Notifikasi Suara Otomatis (Real-Time):** 
  Setiap kali ada uang masuk dari QRIS (baik dari DANA, BCA, OVO, dll.), HP Bapak/Ibu akan langsung mengeluarkan suara konfirmasi beserta nominalnya (Contoh: *"Dana masuk sebesar Rp30.000"*).
  
- **🛑 Bebas Ribet & Hemat Waktu:**
  Bapak/Ibu tidak perlu lagi bolak-balik membuka layar HP atau cek m-banking di depan pembeli. Cukup dengarkan suaranya, lalu lanjut melayani pelanggan.

- **📊 Pantau Omzet Lebih Mudah:**
  Aplikasi dilengkapi halaman dashboard untuk melihat total QRIS masuk hari ini, grafik tren transaksi, serta riwayat pembayaran yang sangat rapi.

- **💸 100% Gratis Tanpa Biaya Sewa:**
  Tidak perlu membayar biaya sewa alat bulanan seperti soundbox fisik pada umumnya.

## 🚀 Cara Download & Install

1. Buka menu **[Releases](https://github.com/ajiputra001/AJIPUTRA-PROJECT/releases/latest)** di halaman GitHub ini.
2. Pada bagian Assets, download file bernama **`app-release.apk`**.
3. Setelah didownload, buka file tersebut dan tekan **Install** (Jika diminta izin *Install dari sumber tidak dikenal/Unknown Sources*, silakan diaktifkan).
4. Buka aplikasi Voice-Notf dan ikuti petunjuk untuk memberikan izin akses agar aplikasi bisa membaca notifikasi uang masuk.
